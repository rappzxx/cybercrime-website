Penjelasan DDoS

Serangan DDoS (Distributed Denial of Service) membuat server overload dengan trafik palsu dari ribuan perangkat (botnet).

Cara Kerja DDoS:

  1. Malware menginfeksi banyak perangkat (bot).
  2. Bot dikendalikan hacker dari jauh.
  3. Bot mengirim trafik berlebihan ke target.
  4. Server tidak sanggup menangani dan lumpuh.
